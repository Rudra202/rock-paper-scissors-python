# 🎮 Rock Paper Scissors Game (Python + Tkinter)

A simple and interactive **Rock Paper Scissors** game built using **Python (Tkinter GUI)** where the player competes against the computer.

---

## 🚀 Features

- 🧑 Player vs 🤖 Computer gameplay
- 🎯 First to **5 points wins**
- ⚡ Fast round updates (2-second delay)
- 🔁 Reset button to restart the game
- 🎨 Clean UI with grey theme
- 🖤 Stylish buttons (always black with white text)
- 📊 Live score tracking

---

## 🛠️ Tech Stack

- **Python 3**
- **Tkinter (GUI)**
- **Random module (for computer moves)**
